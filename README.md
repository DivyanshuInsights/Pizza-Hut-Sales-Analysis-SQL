📌 Pizza Hut Sales Analysis Using SQL
📊 Project Overview

The Pizza Hut Sales Analysis Using SQL project is an end-to-end data analytics case study that explores sales performance and customer behavior using structured datasets. The objective of this project is to extract meaningful business insights using SQL by analyzing transactional data stored in Excel files and structured into a relational database.

This project demonstrates proficiency in data analysis, database design, and advanced SQL techniques, making it an ideal addition to a Data Analyst portfolio.

🎯 Objectives
Design and manage a relational database using real-world datasets.
Analyze sales data to uncover trends and patterns.
Derive actionable insights for business decision-making.
Strengthen SQL skills through practical implementation.
Showcase expertise in data analytics for professional growth.
🛠️ Tools and Technologies
Tool:-	Purpose
SQL (PostgreSQL):-	Data querying and analysis
Microsoft Excel:-	Data storage and preprocessing
🗂️ Dataset Description

The project uses multiple Excel sheets representing different aspects of Pizza Hut’s operations.

Table Name,	Description
orders_time,	Contains order IDs, dates, and times
order_details,	Includes quantity and pizza IDs for each order
pizzas,	Stores pizza sizes and prices
pizza_types,	Provides pizza names, categories, and ingredients
